# Documentacao do Fichario Academico

Esta pasta guarda documentacao operacional e tecnica em Markdown.

- `admin/application_requirements.md`: requisitos para rodar e manter a aplicacao.
- `admin/admin_documentation.md`: funcionamento da area administrativa de documentacao.
- `developer/developer_guidelines.md`: orientacoes de desenvolvimento e manutencao.

Os arquivos desta pasta podem ser versionados no Git. A area administrativa de documentacao edita estes arquivos em Markdown diretamente pela aplicacao.

Nao inclua senhas, tokens, chaves de API, credenciais de banco, dumps da base ou dados pessoais sensiveis.
