## Organizacao

- Documentos gerais da plataforma ficam em `docs/`.
- Documentos especificos de modulo ficam em `<modulo>/docs/`.
- Pontes historicas devem ser mantidas apenas quando houver referencia de compatibilidade.
- Arquivos Markdown nao devem conter segredos e sao bloqueados para acesso HTTP direto pelos `.htaccess`.
