# Identidade visual CAREX

Este arquivo e uma ponte local para a documentacao visual central da Plataforma RENAST.

Na fase atual, o CAREX nao possui paleta, fonte ou componentes visuais proprios. As telas devem seguir Bootstrap vanilla e manter apenas a logo como ativo institucional quando necessario.

Fonte de verdade:

- `../../docs/bootstrap-first-planejamento.md`
- `../../docs/bootstrap-first-exemplos.md`
- `../../docs/tema-css-bootstrap-modulos.md`
- `../../docs/documentacao-visual-centralizada.md`

Qualquer nova personalizacao visual deve ser planejada primeiro na documentacao central, validada em tema claro e registrada como requisito para uma futura derivacao de tema escuro antes de chegar ao CSS.
