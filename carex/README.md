# Carex-BR

## Documentacao visual e tema

As regras de tema, CSS, Bootstrap, navbar, botoes, tabelas, progresso e contraste do CAREX ficam centralizadas em:

- `../docs/identidade-visual-ux.md`
- `../docs/tema-css-bootstrap-modulos.md`

Documentos locais do CAREX devem tratar apenas de regras funcionais, conteudo, dados e comportamento especifico do modulo. Paleta, acento vermelho, sumarios, breadcrumbs, barras de progresso e botoes devem herdar do CSS canonico da plataforma.
